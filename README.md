## Abstract

В данной работе решается задача поиска близких по смыслу документов в разноязычных коллекциях на основе инвариантности их внутренних представлений при переводе. Конкретная задача заключается в проверке сохранения собственной размерности текста при переводе и построении индекса документа как последовательности таких размерностей для его фрагментов. Исследование основано на вычислении собственной размерности для разноязычных текстов одной тематики (на материале статей Википедии) и анализе полученных значений. Новизна подхода состоит в том, что собственная размерность впервые рассматривается как устойчивый к переводу идентификатор смыслового содержания текста. В результате работы подтверждена инвариантность собственной размерности при переводе и показана возможность построения на её основе поискового индекса, позволяющего находить тематически схожие документы независимо от языка.
